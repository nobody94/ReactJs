- store: nơi chứa state
- reducers: thực hiện update lại state
- action: nhận function từ dispatch trả về object gửi cho reducer
- dispatch : function gửi đi gồm nhg định nghĩa như: type, payload,...
- subcription: truyền state đã đc update đến các component
- create store -> dispatch 1 function -> 
- provider cung cấp store và quản lý store (viết riêng cho react redux)
- connect(mapStateToProps, mapDispatchToProps)(Counter) 
    + nhận đầu vào 3 thành phần 
    + funtion return function
    + connect map tất cả state và function thành 1 props
    + connect lấy (mapStateToProps, mapDispatchToProps) từ store rồi map vào (Counter) 
- cách map state ra props để sd trg component
const mapStateToProps = state => {
    return {
        ctr: state.counter
        //state lấy từ store
    };
};

const mapDispatchToProps = dispatch => {
    return {
        onIncrementCounter: () => dispatch({type: 'INCREMENT'})
    };
};

export default connect(mapStateToProps, mapDispatchToProps)(Counter);
